# Console Film Search Project 🎬

[English](#english) | [Русский](#русский)

---

## English

**Console Film Search Project** is a lightweight, Python-based CLI application designed for searching movies right from your terminal. It allows users to query film data, interact with a local database, and manage configurations seamlessly through a user-friendly console menu.

### 📁 Project Structure
* `main.py` — The main entry point to start the application.
* `ui.py` — UI module handling the console interface, user inputs, and menus.
* `code.py` — Core logic for processing search queries and handling data.
* `config.py` — Configuration file for API keys, paths, and environment settings.
* `db/` — Directory containing the local database storage.
* `requirements.txt` — List of required external Python packages.

### 🚀 Installation & Setup

1. **Clone the Repository:**
   ```bash
   git clone https://github.com
   cd search_films_project
   ```

2. **Set Up a Virtual Environment:**
   ```bash
   python -m venv venv
   # Linux/macOS: source venv/bin/activate
   # Windows: venv\Scripts\activate
   ```

3. **Install Dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Run the Application:**
   ```bash
   python main.py
   ```

## Русский

**Console Film Search Project** — это легковесное консольное приложение на Python для быстрого и удобного поиска фильмов. Программа позволяет искать информацию о кинокартинах, взаимодействовать с базой данных и управлять настройками через текстовый интерфейс.

### 📁 Структура проекта
* `main.py` — Главная точка входа для запуска приложения.
* `ui.py` — Модуль, отвечающий за консольный интерфейс (меню, ввод/вывод данных).
* `code.py` — Основная логика поиска и обработки данных.
* `config.py` — Конфигурационный файл (настройки API, пути к файлам).
* `db/` — Директория для хранения локальной базы данных.
* `requirements.txt` — Список внешних зависимостей проекта.

   # Windows: venv\Scripts\activate

3. **Установка зависимостей:**
